How to become a Turing student:
1.) Apply online
  Have resume or create a new one
  Take logic tests
2.) If pass then set up interview with instructor
3.) Take interview
  Take more logic tests with instructor and explain methods for solving problems
  Answer questions regarding opinions and beliefs
4.) If accepted then fill out documents
  For loan
  For enrollment
5.) Take Mod 0 training
6.) Start school
